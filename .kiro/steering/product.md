# Product Overview

個人向け家計簿Webアプリケーション。金融機関からの取引データ自動取得とAIによる費目自動分類を特徴とする、開発者本人の個人利用を想定したツール。

## Core Capabilities

- **金融機関自動連携**: Webスクレイピングによる複数金融機関（銀行・証券・カード）からの取引データ自動取得
- **AI費目分類**: Gemini APIを活用した費目推論と、学習されたルールベースの自動分類
- **取引・口座管理**: 手動入力対応、残高追跡、口座種別によるグループ化
- **月次分析**: 収支サマリー、費目別内訳、月別推移のビジュアル表示
- **PWA対応**: モバイル端末でのインストールとオフライン利用

## Target Use Cases

- 複数の金融機関を横断した収支の一元管理
- 取引履歴からの支出パターン分析
- 費目別・月別の支出傾向の把握
- モバイル端末からの外出先での確認

## Value Proposition

- 金融機関ログイン情報を外部サービスに預けずに自己ホスティングで管理
- 手動分類の手間をAIとルール学習で自動化
- シンプルなUIで個人の家計管理に必要十分な機能を提供
